# Claude API Limits

Cursor / VS Code için Cortex API limit takip eklentisi.

**claude.gg** ve **vertex.claude.gg** üzerindeki günlük ve saatlik limit kullanımını status bar'dan takip et.

## Kurulum

1. `.vsix` dosyasını indir
2. Cursor → Extensions → `···` → Install from VSIX
3. İlk açılışta provider ve API key sor

## Kullanım

- Status bar'da anlık limit görünür
- Hover: detaylı kalan süre
- Tıkla: manuel yenile
- 30 saniyede bir otomatik güncellenir
- `Ctrl+Shift+P` → **Claude API Limits: Provider Seç**
- `Ctrl+Shift+P` → **Claude API Limits: API Key Ayarla**
